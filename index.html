POST /analyze

Description: Analyze DNA/RNA sequences for pattern matches.

Parameters:

  • sequence - The DNA/RNA sequence to analyze
  • query - The pattern to search for

Example Request:

curl -X POST https://rna-analyzer.onrender.com/analyze \
  -H "Content-Type: application/json" \
  -d '{"sequence":"AGCTAGCTAGCTAGCT","query":"AGCT"}'

Example Response:

{
  "sequence_length": 16,
  "query": "AGCT",
  "match_count": 4,
  "match_positions": [1, 5, 9, 13]
}
